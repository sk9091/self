FOUR FORFINGER

Final Year Project of Higher-Diploma in Multimedia


In the project, a iOS game for practice classification and communication was built. This is a four-players game. Also, there are 4 game mode.

There are four slider for players in the game screen, the slider top-bottom and left-right are connected be line and two line create one intersect point, player can control the point by moving each slider. The game need all player make cooperate to move the intersect point to finish the game.

Also, this project won the best FYP Award in IVE Multimedia 2015~2017.
